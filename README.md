
# Machine Learning Projects

This repository contains two machine learning projects demonstrating data preprocessing, model building, and evaluation using real-world datasets.



##  Project 1: Classical Machine Learning Models

This project uses two datasets:
- Glass Identification Dataset
- Diabetes Dataset

###  What I did:
- Cleaned and preprocessed data (handled missing values, scaling)
- Applied feature transformation using RBF kernel
- Visualized data using t-SNE (before and after transformation)
- Trained multiple models:
  - Perceptron
  - Logistic Regression
  - Neural Network (MLP)

###  Results:
- MLP achieved highest performance on both datasets

##  Project 2: Deep Learning with Image Data

This project uses the **CelebA dataset** for image classification.

###  What I did:
- Loaded and processed image dataset using TensorFlow
- Built custom data generator for efficient training
- Implemented deep learning model using:
  - VGG16 (pre-trained CNN)
- Performed classification tasks such as:
  - Gender prediction (Male vs Female)

### 🔹 Advanced Work:
- Combined image data with additional metadata (feature engineering)
- Built two models:
  - Image-only model
  - Image + metadata model
